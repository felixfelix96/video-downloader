## 🚀 部署到 Render

### 第一步：準備 GitHub 倉庫

1. 在 GitHub 創建新倉庫（例如：`video-downloader`）
2. 將 `web_deploy` 文件夾內的所有文件上傳到倉庫根目錄：
   ```
   video-downloader/
   ├── app.py
   ├── requirements.txt
   ├── Procfile
   ├── runtime.txt
   └── templates/
       ├── index.html
       └── login.html
   ```

### 第二步：在 Render 創建服務

1. 登錄 [Render](https://render.com)
2. 點擊 "New +" → "Web Service"
3. 連接你的 GitHub 倉庫
4. 填寫配置：
   - **Name**: `video-downloader`（隨意）
   - **Environment**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app`
5. 點擊 "Create Web Service"

### 第三步：完成

等待部署完成（約 2-5 分鐘），Render 會提供一個免費的 `.onrender.com` 網址。

## ⚠️ 常見問題

### 1. "Requested format is not available" 錯誤
- 已修復：增加了多種格式回退機制
- 如果仍出現，可能是該視頻格式確實不支持

### 2. 視頻太大無法下載
- 免費服務器限制 50MB，大視頻會自動失敗
- 建議下載短視頻或片段

### 3. 抖音/快手提取失敗
- 確保使用分享鏈接（在 App 中點分享複製鏈接）
- 短鏈接（如 `v.douyin.com/xxxxx`）通常比長鏈接更穩定

### 4. Bilibili 某些視頻失敗
- 可能是會員專屬視頻或需要登錄
- 試試其他非會員視頻
