# 視頻下載器 - 本地部署 + 公網訪問指南

## 方案一：使用 ngrok（推薦，最簡單）

### 1. 安裝 Python 依賴

```bash
pip install flask flask-limiter yt-dlp requests
```

### 2. 下載 ngrok

1. 訪問 https://ngrok.com/download
2. 下載 Windows 版本
3. 解壓到任意文件夾

### 3. 註冊 ngrok 賬號

1. 訪問 https://dashboard.ngrok.com/signup
2. 用郵箱註冊（免費）
3. 登錄後獲取 **Authtoken**

### 4. 配置 ngrok

```bash
# 在 ngrok 所在目錄打開 CMD，執行：
ngrok config add-authtoken 你的token
```

### 5. 啟動視頻下載器

```bash
# 在項目根目錄執行
python app.py
```

看到 `Running on http://0.0.0.0:5000` 即可。

### 6. 啟動 ngrok 內網穿透

```bash
# 在 ngrok 目錄打開 CMD，執行：
ngrok http 5000
```

會顯示類似：
```
Forwarding  https://abc123.ngrok-free.app -> http://localhost:5000
```

### 7. 分享給朋友

把 `https://abc123.ngrok-free.app` 發給朋友，他們就能訪問了！

### 1. 下載花生殼客戶端

## 方案三：使用 Cloudflare Tunnel（免費，無流量限制）

### 1. 安裝 cloudflared

```bash
# 下載 Windows 版本
https://github.com/cloudflare/cloudflared/releases
```

### 2. 登錄授權

```bash
cloudflared tunnel login
```

會打開瀏覽器讓你授權。

### 3. 創建隧道

```bash
cloudflared tunnel create video-downloader
```

### 4. 配置隧道

創建配置文件 `config.yml`：

```yaml
tunnel: 你的隧道ID
credentials-file: 認證文件路徑

ingress:
  - hostname: video.yourdomain.com
    service: http://localhost:5000
  - service: http_status:404
```

### 5. 啟動隧道

```bash
cloudflared tunnel run video-downloader
```

## 常用問題

### Q: 電腦關機後還能訪問嗎？
**A:** 不能，必須保持電腦開機且服務運行。

### Q: ngrok 免費版有限制嗎？
**A:** 有，每次重啟域名會變，每月 1GB 流量限制。

### Q: 如何讓域名固定？
**A:** 需要購買 ngrok 付費版（$5/月），或使用花生殼付費版。

### Q: 為什麼本地部署可以下載 Bilibili？
**A:** 因為你的電腦在中國，沒有地區限制，可以正常訪問 Bilibili。

## 快速啟動腳本（Windows）

創建 `start.bat`：

```bat
@echo off
echo 啟動視頻下載器...
start python app.py
timeout /t 3
echo 啟動 ngrok...
start ngrok http 5000
echo 請等待 ngrok 顯示綠色 "online" 狀態
echo 然後複製 https://xxx.ngrok-free.app 分享給朋友
pause
```

雙擊運行即可！
